## 🔧 セットアップ

### 必要な環境変数
```bash
DISCORD_TOKEN=your_discord_bot_token
OPENAI_API_KEY=your_openai_api_key  # GPT-4o対応
FIREBASE_SERVICE_ACCOUNT_KEY={"type":"service_account",...}
```

### Railway デプロイ（推奨）
1. GitHubにプッシュ
2. Railway.appでプロジェクト作成
3. 環境変数設定
4. Procfile確認: `worker: python enhanced_main_v2.py`
5. 自動デプロイ開始

### ローカル実行
```bash
pip install -r requirements.txt
python enhanced_main_v2.py
```
